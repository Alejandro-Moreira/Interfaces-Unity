# Juego del Cube

Cube es un juego arcade desarrollado en Unity donde el jugador debe evitar que los cubos que caen desde la parte superior de la pantalla toquen el suelo. El objetivo es sobrevivir el mayor tiempo posible mientras se acumulan puntos y se gestionan vidas limitadas.
Incluye un menú interactivo, música de fondo, sonidos de interfaz y una transición fluida entre escenas.

## Menú principal

El proyecto incluye una interfaz inicial con:
* Botón para iniciar el juego.
* Botón para salir de la aplicación.
* Panel visual con diseño simple y limpio.
* Sonido de clic al presionar botones.
* Música de fondo en Loop.

## Juego principal

El juego consiste en recolectar cubos que caen desde posiciones aleatorias.

Incluye:

* Sistema de spawn basado en posición fija para mantener coherencia con la cámara.
* Cubos físicos visibles y con gravedad.
* Puntuación que aumenta conforme sobrevives.
* Sistema de vidas que disminuye al fallar.
* Reinicio automático o retorno al menú al perder todas las vidas.

## Requisitos

* Unity 2021.3 o superior.
* PC o dispositivo compatible.

## Cómo ejecutar el proyecto

Clona el repositorio:

git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git


* Abre Unity Hub.
* Selecciona Open Project.
* Navega a la carpeta del repositorio.
* Inicia el proyecto.
* Ejecuta desde la escena SampleScene.
